# Backend

## 1. Fantasy Football Trade Tracker

**Description:**  
This is the backend of the application built using Node and Express JS. It provided the middleware between the Frontend and the Database for accesible data retrieval

**Tech Stack:**
- JavaScript
- Render
- Node
- Express

## 2. Installation & Setup

### Prerequisites
Make sure you have the following installed:
- Node.js (v14 or higher recommended)
- npm (comes with Node.js)
- A code editor like Visual Studio Code

---

### Steps to Run the Backend

1. **Clone the repository**
```bash
git clone https://github.com/your-username/your-repo-name.git
cd backend
npm install
npm start
```

## 3. API Documentation

### Players
- Get/Players - /api/players
- Get/Player - /api/players/(id)
- Post/Player - /api/players/(json-body)
- Put/Player - /api/players/(json-body)

### Users
- Get/Users - /api/users
- Get/User - /api/users/(id)

### Trades
- Get/Trades - /api/trades
- Get/Trade - /api/trades/(id)

### Trade Info
- Get/TradeInfos - /api/trade_info
- Get/TradeInfo - /api/trade_info/(id)
